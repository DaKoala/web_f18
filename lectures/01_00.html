Starter Tags

Tag Description
<!DOCTYPE>  Defines the document type
<html> Defines an HTML document
<head> Defines information about the document
<title> Defines a title for the document
<body> Defines the document's body

Starter Tags

<h1> to <h6> Defines HTML headings
<p> Defines a paragraph
<br> Inserts a single line break
<hr> Defines a thematic change in the content
<!--...--> Defines a comment

What about the style tag?

                
                <style>
                    h1 {color:red;}
                    p  {color:blue;}
                </style>

<style>

  • Defines style information for an HTML document
  • Uses CSS to define those styles
    • h1 {color:red;}

CSS

  • Controls the layout
  • Can be added to HTML many ways
  • Internal: style tag
  • External: stylesheets
  • Cascading Style Sheets

Internal example

                
<!DOCTYPE html>
<html>
<head>
    <style>
        h1 {color:red;}
        p  {color:blue;}
    </style>
</head>
<body>
    <h1>Hi, header</h1>
    <p>This is some text.</p>
</body>
</html>

External example

                
<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Hi, header</h1>
    <p>This is some text.</p>
</body>
</html>

Cascade example

                
<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="styles.css">
    <link rel="stylesheet" href="stylesonstyles.css">
</head>
<body>
    <h1>Hi, header</h1>
    <p>This is some text.</p>
</body>
</html>
